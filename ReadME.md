# Nelian Client

This repository contains the source code for the custom classes developed and modified specifically for **Nelian Client**.

> **Important:** This repository does **not** contain the complete source code of Minecraft or its original source files. It only includes classes that were created, modified, or adapted specifically for Nelian Client.

## Repository Scope

The files in this repository are intended to showcase the custom development and modifications made for Nelian Client, including client-side features, optimizations, rendering improvements, and other custom systems.

The repository does not represent the full Minecraft source code.
